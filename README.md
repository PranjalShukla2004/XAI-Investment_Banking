# XAI-Investment_Banking

Parameters/Statement data being used -

These are more stable than raw magnitudes and work across companies:

Liquidity

Current ratio = current_assets / current_liabilities

Quick ratio = (cash + receivables) / current_liabilities

Working capital / total assets

Leverage & solvency

Debt / equity

Liabilities / assets

Interest coverage = EBIT / interest_expense

Net debt = total_debt − cash

Profitability

Gross margin = gross_profit / revenue

Operating margin = operating_income / revenue

Net margin = net_income / revenue

ROA = net_income / total_assets

ROE = net_income / equity

Cash generation

FCF margin = FCF / revenue

CFO / net_income (earnings quality)

Capex / revenue
